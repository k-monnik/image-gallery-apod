# React Image Gallery
Responsive web app featuring the Pixabay Image API.

## Built with
[React.js](https://reactjs.org/) and [Tailwind CSS](https://tailwindcss.com/)

## Features
Each image has multiple tags, enabling the user to search images by keywords.
